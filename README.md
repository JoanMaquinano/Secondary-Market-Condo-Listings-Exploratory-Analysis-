# Secondary-Market-Condo-Listings-Exploratory-Analysis-
Secondary‑market real estate data is often fragmented, inconsistent, and difficult to aggregate. Unlike primary markets, publicly available listings frequently lack standardization, transaction history, and complete attribute coverage—posing challenges for price comparison and valuation analysis.
This project explores what can (and cannot) be reasonably inferred from available-for-sale listings alone, using a constrained and realistic scope.

Objective
To construct a clean, analyzable dataset of secondary‑market condominium units in Makati within a 1–3M PHP budget, and assess whether available listing data is sufficient to support value‑based decision‑making.

Scope

~30 condominium developments in Makati
Only currently listed units for sale
Publicly available listings collected manually
No proprietary, insider, or transaction‑level data


Methodology

Manual collection of property listings from public sources
Cell‑by‑cell validation to ensure consistency and accuracy
Standardization of key attributes (e.g., price, size, development)
Identification of missing fields, structural inconsistencies, and comparability issues


Key Observations

Listing data varies significantly in completeness and structure across developments
Absence of transaction history limits fair‑value conclusions
Asking prices alone are insufficient to infer pricing efficiency or investment quality
Data preparation is a critical prerequisite before any valuation or optimization analysis


What This Project Does Not Do

It does not attempt price prediction or valuation modeling
It does not provide investment recommendations
It does not claim market‑wide representativeness

The focus is intentionally on data readiness and analytical boundaries, not outcomes.
